# ⏰ Digital Alarm Clock

A simple command-line Digital Alarm Clock built with Python. This project allows users to set an alarm that plays a sound at the specified time. It was created to practice working with Python's `time`, `datetime`, and `pygame` modules.

## Features

- Set an alarm for a specific time
- Plays an alarm sound when the time is reached
- Simple and easy-to-use command-line interface
- Uses real-time clock functionality

## Technologies Used

- Python 3
- `time`
- `datetime`
- `pygame`

## Project Structure

```
digital-alarm-clock/
│── main.py
│── alarm.mp3
└── README.md
```

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Himanshu712008/digital-alarm-clock.git
```

2. Navigate to the project folder:

```bash
cd digital-alarm-clock
```

3. Install the required package:

```bash
pip install pygame
```

## Run the Project

```bash
python main.py
```
